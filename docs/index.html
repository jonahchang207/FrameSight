Capabilities

A lean Windows stack — local inference, local overlay, no cloud required.

⚡

High-FPS capture

DXGI desktop duplication via dxcam. Stale frames are dropped so inference always uses the latest screen.

🎯

YOLO11n detection

Ultralytics nano model tuned for speed. Train on custom classes — enemy and enemy_head included.

🪟

Win32 overlay

Topmost, transparent, click-through window with per-class colors, labels, and confidence scores.

🔧

Train locally

PowerShell setup and training scripts. Weights save to weights/best.pt — one command to run live.

📊

Async pipeline

Separate capture, inference, and overlay threads with live FPS stats in the terminal.

♿

Assistive-first

Built for accessibility research — visual cues, cueing, and HCI studies on live interfaces.
